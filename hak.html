📊 إحصاء وصفي (Statistique Descriptive)

ملخص شامل للفصلين الأول والثاني: المفاهيم الأساسية، الجداول، التمثيلات البيانية، والمقاييس العددية (النزعة المركزية والتشتت).

المصطلحات بالفرنسية مع الشرح بالعربية — تصميم أنيق بألوان وردية وزرقاء

📘 الفصل الأول : أساسيات ومعجم إحصائي (Notions de base)

المجتمع Population و العينة Échantillon

concepts clés

Population (Ω) : المجموعة الكلية المدروسة، عدد أفرادها N (effectif total).
Individu : عنصر من المجتمع.
Échantillon : مجموعة جزئية من المجتمع.
Caractère / Variable : الخاصية التي تُلاحظ أو تُقاس على الأفراد.

\( \Omega = \{ \text{أفراد} \}, \quad N = \sum n_i \)

أنواع المتغيرات Types de caractères

نوعي (Qualitatif) : قيم غير قابلة للقياس العددي مثل الجنسية، اللون.
كمي (Quantitatif) : قابل للقياس.
→ متقطع (Discret) : قيم منعزلة (أعداد صحيحة).
→ مستمر (Continu) : أي قيمة في مجال (ℝ).

مثال : عدد الأطفال (متقطع)، الوزن (مستمر).

الجداول الإحصائية (Tableaux statistiques)

Effectif \(n_i\) : عدد الأفراد الحاملين للقيمة \(x_i\).
Fréquence \(f_i\) : \(f_i = n_i / N\)، و \(\sum f_i = 1\).
Effectifs cumulés croissants (ECC) : \(E_i = \sum_{k=1}^i n_k\).
Fréquence cumulée (FCC) : \(F_i = \sum_{k=1}^i f_k\).

\(N = \sum n_i \quad ; \quad f_i = \frac{n_i}{N} \quad ; \quad F(x) = P(X \le x)\)

المتغير المستمر Variable continue و Classes

تُجمع القيم في فئات \([a_i, a_{i+1}[\):
• Centre de classe \(c_i\) = \(\frac{a_i + a_{i+1}}{2}\)
• Amplitude \(e_i\) = \(a_{i+1} - a_i\)
• Effectif \(n_i\) = عدد القيم في الفئة.
✨ عدد الفئات التقريبي : \(k \approx \sqrt{N}\).

المدرج التكراري (Histogramme) : الارتفاع = \( \frac{n_i}{e_i} \) إذا كانت الأطوال مختلفة.

التمثيلات البيانية (Représentations graphiques)

النوعي (Qualitatif) : مخطط دائري (Camembert)، أعمدة (Diagramme en barres).
الكمي المتقطع : مخطط أسياخ (Bâtons)، مضلع التكرارات.
الكمي المستمر : مدرج تكراري (Histogramme)، مضلع التكرارات المتجمعة (منحنى متدرج).

📐 زاوية القطاع الدائري : \(\alpha_i = 360 \times f_i\)

دالة التوزيع Fonction de répartition \(F(x)\)

لكل \(x \in \mathbb{R}\)، \(F(x) = P(X \le x)\). وهي دالة متدرجة (حالة متقطع) أو متصلة متعددة التعريف (حالة مستمر).
خواص : \(F(-\infty)=0,\; F(+\infty)=1\)، غير تناقصية.

\(F(x) = \sum_{x_i \le x} f_i\)
📈 الفصل الثاني : التمثيل العددي للبيانات (Représentation numérique)

📌 المتوسط الحسابي Moyenne arithmétique

للسلسلة المتقطعة : \(\bar{x} = \frac{1}{N}\sum n_i x_i = \sum f_i x_i\)
للمتغير المستمر : نستخدم مراكز الفئات \(c_i\).
الخاصية : \(y = a x + b \Rightarrow \bar{y} = a\bar{x}+b\).

\(\bar{x} = \frac{\sum n_i x_i}{N}\)
مثال : علامات {5,7,8,10} → \(\bar{x}=7.5\)

📌 الوسيط Médiane (Me)

متقطع : القيمة المركزية بعد الترتيب (N فردي : \(x_{(N+1)/2}\)، زوجي : متوسط القيمتين المركزيتين).
مستمر : استيفاء خطي داخل الفئة الوسيطية.

\(Me = x_k + \frac{\frac{N}{2} - E_{k-1}}{n_k} \cdot (x_{k+1} - x_k)\)

حيث \(E_{k-1}\) التكرار المتجمع قبل الفئة الوسيطية.

📌 المنوال Mode (Mo)

القيمة الأكثر تكراراً (أكبر effectif). للمتغير المستمر نحدد الفئة المنوالية ثم:

\(Mo = x_k + \frac{n_k - n_{k-1}}{(n_k - n_{k-1}) + (n_k - n_{k+1})} \cdot e_k\)

\(n_k\) : تكرار الفئة المنوالية، \(e_k\) طول الفئة.

📌 التباين Variance و الانحراف المعياري Écart-type

Variance : متوسط مربع الانحرافات عن المتوسط.
\(\sigma^2 = \frac{1}{N}\sum n_i (x_i - \bar{x})^2 = \sum f_i (x_i - \bar{x})^2\)
نظرية كونيغ (Théorème de König) لحساب أسهل :

\(\sigma^2 = \frac{1}{N}\sum n_i x_i^2 - (\bar{x})^2\)

الانحراف المعياري : \(\sigma = \sqrt{\sigma^2}\)
خاصية : \(\sigma_{aX+b} = |a|\sigma_X\)

الربيعات Quartiles و المدى Étendue

Étendue : \(e = x_{\max} - x_{\min}\).
Premier quartile \(Q_1\) : \(F(Q_1)=0.25\)
Troisième quartile \(Q_3\) : \(F(Q_3)=0.75\)
الفرق الربيعي \(Q_3 - Q_1\) يحتوي 50% من البيانات.
حساب مستمر : \(Q_1 = x_k + \frac{N/4 - E_{k-1}}{n_k} e_k\)

\(F(Q_1) = \frac{1}{4}, \quad F(Q_3) = \frac{3}{4}\)

معامل الاختلاف Coefficient de variation

CV \(= \frac{\sigma}{\bar{x}}\) (بدون وحدة، يقارن التشتت).
المتوسط الهندسي (Moyenne géométrique) : \(G = \sqrt[N]{\prod x_i^{n_i}}\)
المتوسط التوافقي (Moyenne harmonique) : \(H = \frac{N}{\sum n_i/x_i}\).

💡 CV صغير → بيانات متجانسة، CV كبير → تشتت عالٍ.

قوانين أساسية (Lois essentielles)

🔹 مجموع الانحرافات عن المتوسط : \(\sum n_i (x_i - \bar{x}) = 0\)
🔹 التباين هو القيمة الصغرى لـ \(\sum n_i (x_i - a)^2\) عندما \(a = \bar{x}\).
🔹 تغيير الأصل والمقياس : \(Var(aX+b)=a^2 Var(X)\)
🔹 المتغير المركز : \(X_c = x - \bar{x} \Rightarrow \bar{X_c}=0\)

\( \sum n_i (x_i - \bar{x}) = 0 \)

جدول ملخص القوانين

  • المتوسط : \(\bar{x} = \frac{\sum n_i x_i}{N}\)
  • التباين (كونيغ) : \(\sigma^2 = \frac{\sum n_i x_i^2}{N} - (\bar{x})^2\)
  • الوسيط (مستمر) : \(Me = x_k + \frac{N/2 - E_{k-1}}{n_k} e_k\)
  • المنوال (مستمر) : \(Mo = x_k + \frac{n_k - n_{k-1}}{(n_k - n_{k-1})+(n_k - n_{k+1})} e_k\)
  • التكرار النسبي المتجمع : \(F_i = \sum_{k=1}^i f_k\)

مثال تطبيقي كامل

سلسلة : علامات {10,12,13,13,15,16,18} (N=7)
المتوسط \(\bar{x}=13.86\)، الوسيط = 13، المنوال = 13، التباين ≈ 6.12، \(\sigma\approx2.47\).
للمتغير المستمر (فئات الرواتب) : نستخدم مراكز الفئات ثم المتوسط الموزون.

تطبيق الاستيفاء الخطي للحصول على الوسيط والمنوال بدقة.
